# Project: Weather Javascript Code

## Description

Purpose of this code is to create an application where an array of inputs (location name, postal code); current weather and time can be logged for those locations.

Example: "./weather New York, 10005, Tokyo, São Paulo, Pluto"

## Resources

The application was created using plain JavaScript and requires node installed in Operating System. These files can be cloned from [github link]( https://github.com/adeelbarki/weather-javascript-code.git ) using this command

`$ git clone https://github.com/adeelbarki/weather-javascript-code.git`

Make sure to `$ cd weather-javascript-code` in terminal to access all files.

Files include are :
*   weather
*   README.md
*   test-cases.xlsx

## Run the code

if you have successfully installed node then in a terminal write `node  -v`  to check the version. 

### For linux ubuntu user:

Simply run in the terminal `./weather New York, 10005, Tokyo, São Paulo, Pluto`to run this application. (Keep in mind the directory must be the same as project directory)

### For windows user:

simply run in the command line `node weather New York, 10005, Tokyo, São Paulo, Pluto` to run this application. (Keep in mind the directory must be the same as project directory)

## Testing

Test case file is designed in excel to see what outcomes are needed and the expected results of these outcome if successfull.
